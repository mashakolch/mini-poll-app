Описание функционала:
1.	Опрос из 5 фиксированных вопросов о кофейных привычках
2.	Типы вопросов:
o	Radio buttons (один вариант)
o	Select dropdown (один вариант)
o	Checkboxes (несколько вариантов)
o	Range slider (числовое значение)
3.	Сохранение результатов в памяти сервера
4.	Просмотр статистики по ответам всех участников
5.	Валидация данных 

Интерфейс:
1.	Вкладка "Пройти опрос" - форма с вопросами
2.	Вкладка "Результаты" - список всех сохраненных ответов
3.	Вкладка "Статистика" - графики и цифры по всем ответам
